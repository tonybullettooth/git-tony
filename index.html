  • git clone - path to repository
  • git status - статус текущего локального репозитория
  • git add index.html - добавление нового файла под контроль гита
  • git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарий
  • git commit -am "added push-pull instructions to html"
  • git push - отправить все законченные изменения в репозиторий на сервер
  • git pull - забрать свежие обновления с сервера
